## Tecnologias utilizadas

- [NodeJS](https://nodejs.org/en/)
- [ReactJS](https://pt-br.reactjs.org/) 
- [React Native](https://reactnative.dev/) 
- [Expo](https://expo.io/) 

## Como utilizar?
Clonar o projeto
```
$ git clone https://github.com/AnaCarol21/be-the-hero.git
```
Instalar as dependências
```
npm install
```
Executar aplicação
```
yarn start
```
