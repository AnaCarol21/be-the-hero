# Be the hero
Aplicação para contribuir com ONGS necessitadas
## Tecnologias utilizadas

- [NodeJS](https://nodejs.org/en/)
- [ReactJS](https://pt-br.reactjs.org/) 
- [React Native](https://reactnative.dev/) 
- [Expo](https://expo.io/) 

## Como utilizar?
Clonar o projeto
```
$ git clone URL
```
Instalar as dependências
```
npm install
```
Executar aplicação
```
yarn start
```
